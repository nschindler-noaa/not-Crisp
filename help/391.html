Help

The Help menu contains the major categories of the Help system.

  • Contents: Opens the table of contents for Help system.
  • Introduction: Opens the Introduction section which provides introductory and background information about the COMPASS model.
  • Getting Started: Opens the Getting Started section which details the features and functions of the COMPASS model.
  • Keyword Glossary: Opens the Keyword Glossary section which is a glossary of terms used in COMPASS.
  • Parameter Glossary: Opens the Parameter Glossary section which is a glossary of parameters found in the two key files used by COMPASS: columbia.desc and base.dat.
  • Technical Details: Opens the Technical Details section which contains descriptions of the three key files used by COMPASS (columbia.desc, base.dat, and flow.archive) and the COMPASS Parser.
  • About CRiSP1: Opens a window displaying the version number of COMPASS currently running.

The Help system window has five navigation functions:

  • Contents: Returns to the Help table of contents page.
  • Back: Displays the previously viewed page. This may be different from the Previous page if you followed a link (blue, underlined text) on a page.
  • Search: Allows you to select a topic from a list of topics in the Help system.
  • Prev: Displays the previous page within the Help system.
  • Next: Displays the next page within the Help system.
