Open

Selecting FileOpen opens a file selection dialog box to read data files into the model. You can use this menu option to open any data file in the proper format, either manually created or saved from COMPASS. You can revert to all default settings and parameter values for COMPASS by using this menu option to open a Yearly Input Data File (.dat). In addition, the Open command is represented by in the Toolbar. Several parameter data files can be read into COMPASS simultaneously by using a Control File.
